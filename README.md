# java-web-maven-quarkus-hello-world

## Description
A POC for quarkus framework.

## Tech stack
- java
- maven
  - quarkus

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`
